# MyMiles ✈️ - Auditoria Inteligente de Passagens

O **MyMiles** é uma ferramenta de auditoria para comparar valores de voos em milhas e dinheiro, trazendo transparência para o mercado de fidelidade aérea.

## 🚀 Status do Projeto: Fase 1 Concluída
A aplicação já possui um backend em Docker e uma interface visual para processamento de dados técnicos.

## 📝 Histórico de Versões

### [2026-03-14] - Fase 1: Interface e Docker
- **Interface Visual:** Criação do `index.html` com Bootstrap para exibição de tabelas.
- **Backend Robusto:** Configuração do servidor Flask rodando em Docker.
- **Segurança (CORS):** Implementação de permissões para o navegador falar com o Docker.
- **Versionamento:** Projeto oficialmente iniciado e sincronizado com GitHub.
