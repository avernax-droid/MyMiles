# 🎯 MyMiles Sniper - V2.3 (Local Dev)

O **MyMiles Sniper** é um motor de busca e auditoria para passagens aéreas. Ele extrai tarifas em tempo real e calcula o "Teto de Milhas" para identificar oportunidades de arbitragem entre passagens pagantes e emissões com pontos.

## 🚀 Status do Projeto: Fase 1 Concluída
- ✅ Backend em Flask funcional.
- ✅ Engine de cálculo de arbitragem (Sniper) calibrada.
- ✅ Interface com Autocomplete Local (Fim dos erros de busca).
- ✅ Suporte a múltiplos passageiros e classes.

## 🛠️ Requisitos de Sistema
- **SO:** Windows 11
- **Processador:** Intel Core i7-2600
- **Linguagem:** Python 3.12 (Migrado da 3.14 para maior compatibilidade)
